# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in `actgpr`, please report it
privately via [GitHub Security Advisories](https://github.com/AlxndrSchroeder/actgpr/security/advisories/new)
rather than opening a public issue. This keeps the details private until a
fix is available.

You can expect an initial response within a few days. Once a fix is ready,
a new version will be released and the advisory disclosed.

## Supported Versions

`actgpr` is pre-1.0 research software; only the latest tagged release is
supported with security fixes.
